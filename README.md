Система позволяет управлять базой данных соискателей и формировать отчеты с использованием многокритериальной сортировки.
Функционал
1. **Просмотр базы данных** - отображение всех записей
2. **Добавление записей** - ввод новых соискателей
3. **Удаление записей** - удаление устаревших данных
4. **Редактирование записей** - изменение информации
5. **Формирование отчетов:**
   - Отчет 1: Сортировка по специальности и фамилии
   - Отчет 2: Фильтрация по специальности
   - Отчет 3: Фильтрация по диапазону оклада
